# 🛡️ TrustLine Insurance

A modern and responsive online insurance portal built with **React**, **Vite**, and **Tailwind CSS**. TrustLine Insurance provides users with an intuitive platform to explore, compare, and manage multiple insurance plans through a clean and user-friendly interface.

## 🚀 Features

- 🏥 Health Insurance
- ❤️ Life Insurance
- 🚗 Motor Insurance
- ✈️ Travel Insurance
- 🏠 Home Insurance
- 💼 Business Insurance
- 📊 Compare Insurance Plans
- 💰 Premium Calculator
- 📄 Online Claims
- ❓ Frequently Asked Questions
- 📱 Fully Responsive Design
- 🎨 Modern UI/UX
  
## 🛠️ Tech Stack

### Frontend
- React.js
- Vite
- Tailwind CSS
- JavaScript (ES6+)
- HTML5
- CSS3

### Icons & Animation
- Lucide React
- Motion
